队列的定义和基本运算

Posted on 2019-10-15 | In 队列

队列的定义

队列是模仿人类排队的法则构建的一种数据结构。队列在计算机系统里比比皆是。

队列（Queue），是限定只能在一端插入，另一端删除的线性表。允许删除的一端叫做队头（front），允许插入的一端叫做队尾（rear）。队列也是一种线性表。

对列的插入操作称为“入队”，删除操作称为“出队”。具有先进先出的特性。

普通顺序队在出队和入队的操作中会出现“假溢出”的现象。为了解决这个问题，可以将数组的首尾在逻辑上连接起来，这样就可以循环利用队列里的空间。称为循环顺序队列。

重点！！！！！！！

在循环顺序队列中，我们可以用摸运算来表示队尾指针的位置。

rear = (rear + 1) % MAXLEN;

同理

front = (front + 1) % MAXLEN;

由于循环顺序队列无法判断是否为队满还是对空，所以设置一个空位置，使front 指向它。

则队空 front == rear;

队满 front == (rear + 1)%MAXLEN; / /rear的下一个和front重合了才代表队满

取元素时front要加一：x = seqQueue.data[(seqQueue.front+1)%MAXLEN];
